# @minininja/engine

A simple zero-depencency engine for bootstrapping games.

-----

✅ `Scene`: class

✅ `GameObject`: class

✅ `Script`: class

and one helper

✅ `Vector3`: class

-----

Changes from `Scene` / `GameObject` propagate to its children

All constructors eat a `JSON` with parameters:

✅ `name`: string

✅ `active`: boolean

and four handlers:

✅ `onEnable`: function

✅ `onDisable`: function

✅ `onStart`: function

✅ `onUpdate`: function
